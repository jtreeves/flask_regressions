Maxima

  • Turning points where the graph reaches a relative maximum value before decreasing
  • First coordinate of the point is the x-value where the graph achieves the maximum; second coordinate is the maximum value itself
  • Some functions have multiple points at which they achieve the same maximal value (e.g., sinusoidal), while others have only one maxima (e.g., cubic), and others have none (e.g., linear)
