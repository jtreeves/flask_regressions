How to Use the Returned Object

{% include 'snippets/response_object.html' %}
  • title: The original string you included when you created the collection; this represents the title of the collection
  • independent: The original string you indicated when you created the collection; this represents the independent variable of the collection
  • dependent: The original string you indicated when you created the collection; this represents the dependent variable of the collection
  • precision: The original number you provided when you created the collection to determine how to round the results
  • data_set: The original list of lists you provided when you created the collection; the inner lists constitute the points analyzed
  • source: The original string you provided when you created the collection; this is the collection's unique identifier among all your existing collections
  • date: String representation of when the collection was either created or last updated
  • best_fit: String indicating which regression model has the highest correlation coefficient (e.g., quadratic, sinusoidal)
  • linear_coefficients: Array of two numbers serving as the coefficients of the linear equation that best fits the data set
  • linear_points: Object of arrays of numbers representing different key points on the graph of the linear equation indicated by linear_coefficients (roots will contain one point, while the other three will be null)
  • linear_correlation: Number representing the correlation coefficient of the linear equation indicated by linear_coefficients
  • quadratic_coefficients: Array of three numbers serving as the coefficients of the quadratic equation that best fits the data set
  • quadratic_points: Object of arrays of numbers representing different key points on the graph of the quadratic equation indicated by quadratic_coefficients (inflections will contain null; either maxima or minima will contain one point; roots may contain either zero, one, or two points)
  • quadratic_correlation: Number representing the correlation coefficient of the quadratic equation indicated by quadratic_coefficients
  • cubic_coefficients: Array of four numbers serving as the coefficients of the cubic equation that best fits the data set
  • cubic_points: Object of arrays of numbers representing different key points on the graph of the cubic equation indicated by cubic_coefficients (inflections will contain one point; maxima and minima will either both contain one point or both contain null; roots may contain either one, two, or three points)
  • cubic_correlation: Number representing the correlation coefficient of the cubic equation indicated by cubic_coefficients
  • hyperbolic_coefficients: Array of two numbers serving as the coefficients of the hyperbolic equation that best fits the data set
  • hyperbolic_points: Object of arrays of numbers representing different key points on the graph of the hyperbolic equation indicated by hyperbolic_coefficients (roots will contain one point, while the other three will be null)
  • hyperbolic_correlation: Number representing the correlation coefficient of the hyperbolic equation indicated by hyperbolic_coefficients
  • exponential_coefficients: Array of two numbers serving as the coefficients of the exponential equation that best fits the data set
  • exponential_points: Object of arrays of numbers representing different key points on the graph of the exponential equation indicated by exponential_coefficients (all will be null)
  • exponential_correlation: Number representing the correlation coefficient of the exponential equation indicated by exponential_coefficients
  • logarithmic_coefficients: Array of two numbers serving as the coefficients of the logarithmic equation that best fits the data set
  • logarithmic_points: Object of arrays of numbers representing different key points on the graph of the logarithmic equation indicated by logarithmic_coefficients (roots will contain one point, while the other three will be null)
  • logarithmic_correlation: Number representing the correlation coefficient of the logarithmic equation indicated by logarithmic_coefficients
  • logistic_coefficients: Array of three numbers serving as the coefficients of the logistic equation that best fits the data set
  • logistic_points: Object of arrays of numbers representing different key points on the graph of the logistic equation indicated by logistic_coefficients (inflections will contain one point, while the other three will be null)
  • logistic_correlation: Number representing the correlation coefficient of the logistic equation indicated by logistic_coefficients
  • sinusoidal_coefficients: Array of four numbers serving as the coefficients of the sinusoidal equation that best fits the data set
  • sinusoidal_points: Object of arrays of numbers representing different key points on the graph of the sinusoidal equation indicated by sinusoidal_coefficients (string values will be provided for general forms where applicable)
  • sinusoidal_correlation: Number representing the correlation coefficient of the sinusoidal equation indicated by sinusoidal_coefficients
