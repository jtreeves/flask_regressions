Equations

All functions can be created by using the numbers provided in the coefficients array. For example, if the linear_coefficients array is [7.64, 3.41], then that means the a value of the linear equation is 7.64 and the b value of the linear equation is 3.41.

  • Linear

    • Description

      Straight line with a constant slope and a single x-intercept

    • Application

      Best suited to situations with constant growth

    • General Form

      y = a\cdot{x} + b

    • Specific Example

      y = 2\cdot{x} + 3

  • Quadratic

    • Description

      Parabola with either a maximum or a minimum

    • Application

      Best suited to situations where you hope to maximize or minize a specific variable

    • General Form

      y = a\cdot{x^2} + b\cdot{x} + c

    • Specific Example

      y = x^2 - 12\cdot{x} + 66

  • Cubic

    • Description

      Curve with potentially both a maximum and a minimum

    • Application

      Best suited to situations where you assume a variable can be maximized at one instance and minimized at another

    • General Form

      y = a\cdot{x^3} + b\cdot{x^2} + c\cdot{x} + d

    • Specific Example

      y = x^3 - 15\cdot{x^2} + 63\cdot{x} - 31

  • Hyperbolic

    • Description

      Curve that approaches an asymptote

    • Application

      Best suited to situations where you assume the output will fall to a certain value over time

    • General Form

      y = a\cdot{\frac{1}{x}} + b

    • Specific Example

      y = 2\cdot{\frac{1}{x}} + 3

  • Exponential

    • Description

      Curve that increases or decreases at a faster rate than any of its peers

    • Application

      Best suited to situations where you assume the output will increase or decrease rapidly and without any limits

    • General Form

      y = a\cdot{b^x}

    • Specific Example

      y = 2\cdot{3^x}

  • Logarithmic

    • Description

      Curve that grows slowly

    • Application

      Best suited to situations where you assume the output will increase at a slower-than-linear rate

    • General Form

      y = a\cdot{\ln{x}} + b

    • Specific Example

      y = 2\cdot{\ln{x}} + 3

  • Logistic

    • Description

      Curve that passes through a period of exponential growth before approaching an asymptote

    • Application

      Best suited to situations where you assume the output will increase rapidly before levelling off

    • General Form

      y = \frac{a}{1 + \text{e}^{-b\cdot(x - c)}}

    • Specific Example

      y = \frac{2}{1 + \text{e}^{-3\cdot(x - 5)}}

  • Sinusoidal

    • Description

      Curve that oscillates back and forth between a maximum and minimum value

    • Application

      Best suited to situations where you expect cyclical behavior

    • General Form

      y = a\cdot{\sin(b\cdot(x - c))} + d

    • Specific Example

      y = 5\cdot{\sin(2\cdot(x - 3))} + 7
