Points

Every graph is made up of an infinite number of points, but only a few of them are important enough to warrant special mention. These points coincide with visual graphical features (e.g., a turning point).

  • Roots

    • Points where the graph crosses the x-axis
    • First coordinate of the point is the x-value where the graph crosses the x-axis; second coordinate is 0 (the y-coordinate of all x-intercepts)
    • Some functions have multiple roots (e.g., cubic, sinusoidal), while others have only one (e.g., linear), and others have none (e.g., exponential)
    • This cubic function has a root at (0.5653, 0.0)

  • Maxima

    • Turning points where the graph reaches a relative maximum value before decreasing
    • First coordinate of the point is the x-value where the graph achieves the maximum; second coordinate is the maximum value itself
    • Some functions have multiple points at which they achieve the same maximal value (e.g., sinusoidal), while others have only one maxima (e.g., cubic), and others have none (e.g., linear)
    • This cubic function has a maximum at (3.0, 50.0)

  • Minima

    • Turning points where the graph reaches a relative minimum value before increasing
    • First coordinate of the point is the x-value where the graph achieves the minimum; second coordinate is the minimum value itself
    • Some functions have multiple points at which they achieve the same minimal value (e.g., sinusoidal), while others have only one minima (e.g., cubic), and others have none (e.g., linear)
    • This cubic function has a minimum at (7.0, 18.0)

  • Inflections

    • Points of inflection where the graph changes its concavity and either stops increasing and starts decreasing or stops decreasing and starts increasing
    • First coordinate of the point is the x-value where the graph changes its concavity; second coordinate is the y-value that happens to occur during this changing of concavity
    • Some functions have multiple inflection points (e.g., sinusoidal), while others have only one inflection point (e.g., cubic), and others have none (e.g., linear)
    • This cubic function has an inflection point at (5.0, 34.0)
